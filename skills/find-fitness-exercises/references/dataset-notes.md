# Dataset notes

## Source and coverage

- Source: `https://github.com/hasaneyldrm/exercises-dataset`
- Default branch: `main`
- Records: 1,324 exercises in `data/exercises.json`
- Searchable metadata: `name`, `category`, `body_part`, `equipment`, `target`, `muscle_group`, and `secondary_muscles`
- Instructions: English, Spanish, Italian, Turkish, Russian, Chinese, Hindi, Polish, Korean, and French
- Reference media: one 180×180 thumbnail and one animation GIF per record

The dataset does not contain difficulty ratings, contraindications, sets, reps, loads, or individualized programming. Do not present inferred values for those fields as dataset facts.
