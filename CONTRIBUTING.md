# Contributing to Exercises Dataset

Thank you for your interest in improving the exercises dataset! We welcome contributions to clean up data, translate instructions into more languages, or improve the developer guides.

## How to Contribute

### 1. Suggesting Data Corrections
If you find a typo, incorrect target muscle, or awkward translation:
- Please open an issue using the **Exercise Correction** template.
- Provide the Exercise ID (e.g. `0001`) and the suggested fix.

### 2. Submitting Code Changes
To improve the exercise browser (`index.html`) or developer setup guide (`setup.html`):
1. Fork the repository.
2. Clone your fork locally.
3. Start a local server:
   ```bash
   npm install
   npm run dev
   ```
4. Make your changes in a new branch and submit a Pull Request.
